# 💄 Loja de Maquiagem - E-commerce Web

Projeto de **loja virtual de maquiagem** desenvolvido com foco em **front-end moderno, carrinho de compras e integração com backend em PHP**.

💄 Loja de Maquiagem – GlamBeauty

Projeto de E-commerce desenvolvido com PHP, MySQL, Bootstrap 5 e Docker, com sistema de:

Cadastro e Login de usuários

Carrinho de compras

Checkout

Painel Administrativo

Cadastro de produtos com upload de imagem

🚀 Tecnologias Utilizadas

PHP 8+

MySQL

Bootstrap 5

HTML5

CSS3

Docker + Docker Compose

Apache

A aplicação simula um **e-commerce completo**, permitindo visualizar produtos, adicionar ao carrinho e realizar checkout com formulário de pagamento.

# 🛒 Funcionalidades

### 🧴 Catálogo de Produtos
- Batom
- Base líquida
- Sombra
- Perfume
- Esmalte
- Máscara de cílios

Cada produto possui:

- imagem
- descrição
- preço
- botão de adicionar ao carrinho

---

### 🛍️ Carrinho de Compras

O carrinho permite:

- adicionar produtos
- alterar quantidade
- remover produtos
- calcular subtotal
- calcular total automaticamente

Tudo atualizado dinamicamente com **JavaScript**.

---

### 💳 Checkout

O sistema possui formulário de finalização de compra contendo:

- Nome
- Email
- Endereço
- Forma de pagamento

Opções de pagamento:

- Cartão de crédito
- PIX
- Boleto

Os dados são enviados para:
checkout.php

---

### 👩‍💻 Sistema de Login

Sistema simples de autenticação com:

- login
- cadastro
- logout

Arquivos responsáveis:
login.php
register.php
logout.php
